Top-level method that parse a Term in general, which may recursively call itself.

There are 5 valid cases:

  1. (Op, A1, ..., An) is a CompoundTerm if Op is a built-in operator
  2. {A1, ..., An} is an SetExt;
  3. [A1, ..., An] is an SetInt;
